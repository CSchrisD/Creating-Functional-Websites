Steps

To find the correct solution of an expression the operations must be done in a specific order. If the steps are not done in this order the correct solution will never be found. The only exception to this rule are multiply and divide can be done in either order, as well as subtract and add can be done in either order, as long as they are done at step 3 and 4 respectively.
