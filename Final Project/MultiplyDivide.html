Multiplication - repeated addition

The commutative property of multiplication

When doing multiplication order does not matter, the final solution will be the same.


a × b = b × a

Example:

5 ×; 3 = 15
3 ×; 5 = 15
5 ×; 3 = 3 × 5


The associative property of multiplication

Finding the solution in the parenthesis is done first, but similar to the commutative property order does not matter. The solution will be the same.


a × (b × c) = (a × b) + c

Example:

4 ×; (2 × 6) = 48
(4 × 2) × 6 = 48
4 × (2 × 6) = (4 × 2) × 6


The distributive property

When multiplying a number outside of the parenthesis, the number can be multiplied with all objects first and the remaining operations can be done.


a(b + c) = ab + ac

Example:

5(8 + 3) = 55
(5 × 8) + (5 × 3) = 55
5(8 + 3) = (5 × 8) + (5 × 3)


Multiplying negatives

When multiplying a negative and a positive the result is negative.


a × -b = -c


When multiplying two negative numbers the result is positive.


-a × -b = c

Example:

5(8 + 3) = 55
(5 × 8) + (5 × 3) = 55
5(8 + 3) = (5 × 8) + (5 × 3)

Division - the number of times a number goes into another

Rules of division

Any number divided by 1 is itself


a ÷ 1 = a


While any number divided by itself is 1


a ÷ a = 1


No number can be divided by 0


a ÷ 0 = Error
