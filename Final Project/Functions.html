Functions

Input

The input is the number that will take the place of the variable in an equation. A function can have any number inserted into the place of the variable.


x = a

Equation

The equation the input is placed in is usually stated as "f of x equals..." and then the remaining equation. The portion f(x) is used to remember the current number being used when finding the final answer.


f(x) = x2


f(2) = 22


f(2) = 4

Output

The output is going to be the final answer after solving the equation with the given input. A function is only a function if every input has exactly one output, but an output can be the same result from different inputs.
